实验目的：熟悉个人博客系统的搭建。
具体包括：
参考方案：注册Github账号，搭建Hexo环境并实现个人博客搭建，撰写实验报告。

这次实验是在实验3.1的基础上进行的，主要是把博客发布到GitHub上去，使用Git和hexo来完成。
主要遇到了以下几个问题：

  1. failed to push some refs to“”：
    这个是用户名和邮箱问题，Git没有登录权限，
    需要在config文件中加入自己的用户名和邮箱。
  2. error：spawn failed…：
    删除.deploy_git文件夹;
    输入git config –global core.autocrlf false
    然后，依次执行：
    hexo clean
    hexo g
    hexo d

结论分析与体会：
通过这次实验，我学会了如何注册Github账号，搭建Hexo环境并实现个人博客搭建，在搭建个人博客的过程中，我学习了vim命令以及git的使用方法，并把自己的博客上传到GitHub中，跟着博客搭建教程的学习也令我受益匪浅，最后也成功搭建了自己的博客。总的来说，这次实验中，我收获颇丰。
